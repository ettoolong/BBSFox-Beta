連線到站台：
例如要連線到站台aaa.bbb.ccc.ddd，
在Firefox網址列輸入telnet://aaa.bbb.ccc.ddd就可以連線，此連線預設使用telnet的連接埠23，
如果站台使用特定的連接埠例如8888，在Firefox網址列輸入telnet://aaa.bbb.ccc.ddd:8888即可。

一般熱鍵：
(熱鍵可在設定頁開啟或關閉)
Ctrl+Shift+V貼上
Ctrl+A全選
圈選文字後按下 Ctrl + C複製
Ctrl+D下載文章到剪貼簿
Ctrl+E顯示/隱藏 BBS背景圖片
Ctrl+M滑鼠瀏覽開／關

滑鼠瀏覽：
(在設定頁設定滑鼠瀏覽以啟用此功能)
一般頁面
頁面左側 = 方向鍵左(返回上一層頁面)
頁面其他部份 = Enter (點擊進入)

在看板列表或文章列表時
參照下圖：
紅色區域 = Home (第一頁)
綠色區域 = End (最後一頁)
藍色區域 = Page Up (上一頁)
黃色區域 = Page Down (下一頁)
橘色區域 = 方向鍵左(返回上一層頁面)
紫色區域 = Enter (進入看板/閱讀文章)

閱讀文章內容時
參照下圖：
綠色區域 = End (最後一頁)
藍色區域 = Page Up (上一頁)
黃色區域 = Page Down (下一頁)
橘色區域 = 方向鍵左(返回上一層頁面)

幾個特別的處理：
圈選某些文字時，滑鼠瀏覽會自動不生效，以免對圈選的文字作動作時誤觸滑鼠瀏覽的動作，
單點畫面取消圈選後，滑鼠瀏覽模式會回復。
滑鼠瀏覽模式下不能用double click去圈選文字(會自動取消圈選)，以免點太快造成文字被圈選，
而下一次的點擊因為圈選了文字而被略過。
使用 ctrl+M去切換滑鼠瀏覽模式，如果需要一般性的操作，
想暫時取消滑鼠瀏覽模式時可以使用此切換熱鍵，以避免操作時誤觸滑鼠瀏覽的動作。

進階滑鼠瀏覽：
(在設定頁設定進階滑鼠瀏覽以啟用此功能)
文章列表或閱讀文章時
畫面左上 = 送出'=' (跳到同標題第一篇)
畫面左上第二/三列 = 送出'[' (跳到同標題上一篇)
畫面右上 = 送出']' (跳到同標題下一篇)
畫面右下 = 跳到同標題最後一篇
畫面左下 = 連續送出 方向鍵左+Enter+End (快速更新推文或顯示新的文章)
